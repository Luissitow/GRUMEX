## 🏭 Descripción del Proyecto

Sitio web corporativo completo para **GRUMEX** (Grupo Comercial Mexicano de la Industria y la Construcción), empresa líder en servicios industriales en México con más de 20 años de experiencia. La plataforma integra 5 líneas principales de negocio con casos de éxito documentados incluyendo clientes como Walmart, CFE, Hyatt y U-Storage.

## ✨ Características Principales

### 🛠️ Tecnologías Utilizadas
- **Frontend**: HTML5 semántico, CSS3/SCSS modular, JavaScript ES6+
- **Librerías**: Leaflet.js (mapas), Font Awesome (iconos)
- **SEO**: Schema.org, Open Graph, metadatos estructurados
- **Responsive**: Mobile-first design con CSS Grid/Flexbox

### 🏢 Servicios Integrados
1. **Manufactura** - Corte láser, soldadura, pintura electrostática, CNC
2. **Mobiliaria** - Mobiliario de oficina, industrial, retail y hogar
3. **Construcción** - Obra civil, remodelaciones, instalaciones especializadas
4. **Mármol** - Cubiertas, revestimientos, mobiliario en mármol
5. **Importación** - Logística, gestión aduanal, maquinaria especializada

### 🎯 Funcionalidades Destacadas
- ✅ **Arquitectura multi-página** (25+ páginas interconectadas)
- ✅ **SEO avanzado** con Schema.org y metadatos estructurados
- ✅ **Responsive design** optimizado para todos los dispositivos
- ✅ **Integración WhatsApp Business** para generación de leads
- ✅ **Geolocalización** con Leaflet.js
- ✅ **Casos de éxito** interactivos con slider corporativo
- ✅ **Formularios de contacto** y cotización
- ✅ **Preloader** y optimización de rendimiento

## 🚀 Instalación y Uso

### Prerequisitos
- Navegador web moderno
- Servidor local (opcional): Live Server, XAMPP, o similar

### Instalación Local
```bash
# Clonar el repositorio
git clone https://github.com/Luissitow/GRUMEX.git

# Navegar al directorio
cd GRUMEX

# Abrir con Live Server o servir con un servidor local
# No requiere instalación de dependencias adicionales
```
